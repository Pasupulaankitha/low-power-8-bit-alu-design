# low-power-8-bit-alu-design
This project implements an 8-bit Low Power Arithmetic Logic Unit (ALU) using Verilog HDL. The ALU performs multiple arithmetic and logical operations based on a 4-bit select signal. A simple low-power technique using an Enable (en) signal is incorporated to reduce unnecessary switching activity when the ALU is idle
